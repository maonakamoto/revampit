# Glossary (Shared Vocabulary)

- Canonical: Offizielle, vorrangige Quelle der Wahrheit für einen Datentyp.
- Authority Order: Reihenfolge, in der Konflikte zwischen Quellen aufgelöst werden.
- Upstream: Informationsquellen, von denen dieses Artefakt abhängt.
- Downstream: Artefakte/Prozesse, die von diesem Artefakt abhängen.
- First Principles: Entscheidungen aus Zielen, Constraints, Invarianten und überprüfbaren Annahmen herleiten.
- RLS (Row Level Security): Datenbank-Sicherheitskonzept (z. B. Supabase/Postgres) für zeilenbasierte Zugriffssteuerung.
- Intake: Prozess der Erfassung/Annahme von Geräten und Metadaten.
- KPI: Key Performance Indicator; zentrale Metrik zur Steuerung.
