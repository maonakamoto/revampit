# Reboot Content Admin Interface

## 🎯 For Non-Technical Team Members

This guide explains how to use the admin interface to manage website content. No technical knowledge required!

---

## 🚀 Getting Started

### 1. Access the Admin Interface
- Visit: `http://localhost:3000/admin` (when running locally)
- Or: `https://yourdomain.com/admin` (when deployed)

### 2. Login
- Enter your email and password
- Click "Sign in"
- You'll be redirected to the admin dashboard

---

## 📄 Managing Pages

### View All Pages
1. After logging in, you'll see the "Pages" section
2. All your website pages are listed here
3. Use the search box to find specific pages
4. Click on any page to edit it

### Create a New Page
1. Click the green "+ New Page" button
2. Fill in the page details:
   - **Page Title**: The main heading (required)
   - **URL Slug**: How the page appears in the URL (required)
   - **Page Content**: Write your content using the editor
   - **SEO Title**: Custom title for search engines (optional)
   - **SEO Description**: Brief description for search engines (optional)

### Edit an Existing Page
1. Click "Edit" next to any page in the list
2. Make your changes in the editor
3. Click "Save Draft" to save without publishing
4. Click "Publish" to make changes live immediately

---

## ✏️ Using the Content Editor

### Basic Formatting
- **Bold**: Select text and click the **B** button
- **Italic**: Select text and click the *I* button
- **Underline**: Select text and click the U button
- **Lists**: Click the bullet (•) or numbered (1.) buttons

### Writing Tips
- Start with a clear headline
- Keep paragraphs short and readable
- Use bullet points for lists
- Save frequently while editing

### Preview Your Work
- Your changes are automatically previewed as you type
- The editor shows exactly how the page will look

---

## 🔄 Publishing Workflow

### Option 1: Save as Draft
- Click "Save Draft"
- Changes are saved but not visible to visitors
- You can come back and edit later

### Option 2: Publish Immediately
- Click "Publish"
- Changes go live immediately
- All website visitors can see the updates right away

---

## 🔐 User Roles

### Editor
- Can create and edit pages
- Can publish content immediately
- Cannot delete pages or manage users

### Admin
- All Editor permissions
- Can delete pages
- Can manage user accounts
- Can access system settings

---

## 🆘 Getting Help

### If You Can't Login
- Contact an Admin to get your login credentials
- Make sure you're using the correct email address

### If You Need Access
- Ask an Admin to create an account for you
- They'll need your email address and preferred role

### If Something Doesn't Work
- Try refreshing the page
- Clear your browser cache
- Contact the development team

---

## 📋 Quick Checklist

Before publishing a page:
- [ ] Page title is clear and descriptive
- [ ] Content is proofread and error-free
- [ ] All links work (if any)
- [ ] Images are properly sized (if any)
- [ ] Page loads correctly in preview

---

## 🎉 You're All Set!

The admin interface is designed to be intuitive and powerful. Start by creating or editing a page, and you'll quickly get the hang of it. Remember: you can always save as draft if you're not ready to publish yet.

Happy content creating! 🚀
